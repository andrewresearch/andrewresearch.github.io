Current Tech Explorations

21 Apr 2016
technologies / programming / web development
markdown / aws / lambda architecture / scala / spark

I thought it was fitting to focus my new site’s first post on the current tech that I’m exploring at the moment.

Firstly, this site uses some tech that I have been using for a while and combines it with some new stuff. All of these posts are written in markdown, which I have been using for a while with readme files on GitHub. I can recommend MacDown as a markdown editor for OS X. What is new however, is that I’m using Hugo, a static site generator to turn the markdown into a static website. However, rather than running this directly on my Mac, I upload my markdown files to an Amazon AWS S3 bucket, and an AWS Lambda service automatically runs Hugo and puts the resulting site into a different S3 bucket. I followed this blog post to set it up, and so far it is working great!

Another tech interest for me at the moment is the Lambda Architecture (not to be confused with the AWS Lambda service or a programming lanugage Lambda function).

I have been programming in Scala for a number of years now. The combination of Scala together with Lambda Architecture and an interest in Machine Learning, has led me to explore Apache Spark and in particular mLib, the machine learning library of Spark.
